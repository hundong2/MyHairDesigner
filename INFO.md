# My Hair Designer (나만의 AI 헤어 디자이너) 💇‍♀️✨

**My Hair Designer**는 최신 Google Gemini AI 기술을 활용하여 사용자의 얼굴형을 분석하고, 가장 잘 어울리는 헤어스타일을 추천하며, 실제 적용 모습을 시뮬레이션해 주는 차세대 뷰티 애플리케이션입니다.

단순한 합성이 아닌, 생성형 AI를 통해 사진관 수준의 고품질 스타일링 이미지를 제공하며, 대화형 AI 스타일리스트와의 상담을 통해 스타일을 미세 조정하고 실제 미용실 예약 정보까지 연결합니다.

---

## 🌟 주요 기능 (Key Features)

### 1. AI 얼굴형 분석 및 맞춤 추천 (AI Face Analysis)
*   **Gemini Vision 멀티모달** 기술을 사용하여 업로드된 사용자 사진(셀카)의 얼굴형(계란형, 둥근형, 각진형 등)을 정밀하게 분석합니다.
*   분석된 얼굴형과 특징을 바탕으로 데이터베이스에 있는 트렌딩 헤어스타일 중 **'Best Match(최적의 스타일)'**를 자동으로 선별하여 시각적으로 강조 표시합니다.
*   **성별 선택(남/여)** 기능을 통해 성별에 특화된 최신 유행 헤어스타일 10종씩을 각각 추천받을 수 있습니다.

### 2. 고화질 가상 헤어 시뮬레이션 (Virtual Try-On)
*   사용자가 선택한 스타일을 **Gemini Imagen** 모델을 통해 실제 사진처럼 자연스럽게 생성합니다.
*   기존 얼굴의 이목구비와 표정을 유지하면서 헤어스타일만 변경하는 고도화된 인페인팅(In-painting) 및 이미지 생성 기술이 적용되었습니다.
*   **Before / After 비교 기능**: 버튼을 누르고 있으면 원본 사진과 스타일링 후 사진을 즉시 비교할 수 있어 변화를 체감하기 쉽습니다.

### 3. 대화형 AI 스타일리스트 (Interactive AI Chat Consultant)
*   결과 화면에서 AI 챗봇과 실시간으로 대화할 수 있습니다.
*   **Function Calling (함수 호출)** 기술 적용: 사용자가 채팅으로 "머리 좀 더 짧게 해줘", "염색해줘", "앞머리 내줘"라고 요청하면, AI가 의도를 파악하고 **'modify_hairstyle'** 도구를 호출하여 즉시 이미지를 수정합니다.
*   스타일링 팁, 관리 방법, 제품 추천 등 전문적인 조언을 받을 수 있습니다.

### 4. 스마트 가이드 및 스타일 카드 (UX & Sharing)
*   **Smart Guide**: 현재 진행 단계(분석 -> 선택 -> 생성)에 맞춰 사용자에게 필요한 팁과 가이드를 제공합니다.
*   **Style Card**: 완성된 스타일, 얼굴형 분석 결과, 매칭 점수가 포함된 세련된 카드를 생성하여 친구들에게 공유할 수 있습니다.

### 5. 위치 기반 미용실 찾기 (Salon Finder via Grounding)
*   **Google Maps Grounding** 기술을 활용하여 사용자의 현재 위치(Geolocation)를 기반으로 해당 스타일을 잘하는 주변 미용실을 검색합니다.
*   단순 검색이 아닌, 별점 순으로 정렬된 상위 3개 미용실의 이름, 주소, 추천 이유를 요약하여 보여줍니다.
*   앱 내에서 구글 지도를 바로 확인할 수 있습니다.

## 🛠 기술 스택 (Tech Stack)

이 앱은 최신 웹 기술과 Google의 Generative AI 생태계를 결합하여 구축되었습니다.

*   **Frontend**: React 19, TypeScript, Tailwind CSS
*   **AI Model**:
    *   `gemini-2.5-flash`: 빠른 응답 속도로 얼굴 분석, 채팅, 미용실 검색(Grounding) 처리에 사용
    *   `gemini-2.5-flash-image`: 고품질의 헤어스타일 이미지 생성 및 변형에 사용
*   **Key AI Features**:
    *   **Multimodal Vision**: 이미지 입력을 텍스트 프롬프트와 결합하여 분석
    *   **Function Calling**: 자연어 명령을 코드로 변환하여 이미지 수정 기능 제어
    *   **Google Search/Maps Grounding**: 최신 위치 정보를 기반으로 실시간 미용실 데이터 제공
*   **Markdown Rendering**: AI의 분석 결과를 가독성 좋게 보여주기 위한 `react-markdown` 활용

## 📱 사용자 경험 흐름 (User Flow)

1.  **사진 업로드**: 메인 화면에서 본인의 셀카를 업로드합니다.
2.  **자동 분석**: AI가 얼굴형을 분석하고 결과를 알려줍니다.
3.  **스타일 선택**: 추천된(초록색 테두리) 스타일 또는 원하는 스타일을 선택합니다.
4.  **AI 생성**: 'Generate Look' 버튼을 눌러 스타일을 적용합니다.
5.  **결과 확인 및 수정**: 변화된 모습을 확인하고, AI 채팅을 통해 추가 수정(염색, 컷트 등)을 요청하거나 관리법을 물어봅니다.
6.  **미용실 찾기**: 마음에 드는 스타일을 시술할 수 있는 근처 미용실 정보를 확인합니다.
7.  **공유하기**: 스타일 카드를 생성하여 SNS에 공유합니다.
